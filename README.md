# ShootingGallery

A simple target shooter game built with **MonoGame**. Practice your mouse skills with a fun, interactive shooting gallery experience.

## 🎯 Features

- Mouse-aim shooting mechanics
- Simple, engaging gameplay
- Built with MonoGame framework

## 📋 Requirements

- .NET Framework / .NET Core (version X.X+)
- MonoGame 3.8+ 
- Windows / macOS / Linux (depending on build)

## 🚀 Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/USMCsky/ShootingGallery_game.git
   cd ShootingGallery_game
   ```

2. Extract the game files:
   ```bash
   unzip ShootingGallery.zip
   ```

3. Run the executable or build from source (if applicable)
